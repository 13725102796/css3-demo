# css3 vue demo

> A Vue.js project

## Introduction
``` bash
  这是一个使用vue来写的demo库，里面有各种css的特效组件，常用的实战开发组件，拿来即用。
  如： swiper scroll infinite toast ....
```
## 插件
  添加了车牌号校验和银行卡校验
## 使用方法

``` bash
# git clone https://github.com/13725102796/css3-demo.git
# cd css3-demo
# npm i
# serve with hot reload at localhost:8080
npm run dev


```
